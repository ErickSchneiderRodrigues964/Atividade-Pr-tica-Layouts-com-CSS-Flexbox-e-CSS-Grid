# 🚀 Exercícios de Layout CSS (Aula 2)

Este repositório contém uma série de **exercícios práticos focados no desenvolvimento de layouts modernos** utilizando **Flexbox** e **CSS Grid**.  
O objetivo foi explorar desde **alinhamentos simples** até **estruturas de páginas mais complexas e responsivas**.

---

# 📂 Estrutura do Projeto

O repositório está organizado por exercícios, cada um focando em uma técnica específica.

## 1️⃣ Exercício 01: Barra de Navegação (Flexbox)

**Foco:** Alinhamento horizontal e distribuição de elementos.

**Técnicas utilizadas:**
- `display: flex`
- `justify-content: center`
- Efeitos de `hover`

---

## 2️⃣ Exercício 02: Grade de Cards (Flexbox Wrap)

**Foco:** Criação de um catálogo de cards flexível.

**Técnicas utilizadas:**
- `flex-wrap: wrap` para permitir que os elementos quebrem linha conforme o tamanho da tela
- `gap` para espaçamento uniforme entre os cards

---

## 3️⃣ Exercício 03: Galeria de Imagens (CSS Grid)

**Foco:** Layouts assimétricos e controle de áreas.

**Técnicas utilizadas:**
- `grid-template-columns` com `repeat(3, 1fr)`
- `grid-column: span 2` para destacar cards específicos

---

## 4️⃣ Exercício 04: Wireframe Complexo (Grid Areas)

**Foco:** Estruturação avançada de componentes de interface.

**Técnicas utilizadas:**
- `grid-template-areas` para organizar:
  - **Header**
  - **Nav**
  - **Aside**
  - **Main**
  - **Footer**

---

## 5️⃣ Exercício 05: Layout de Página Completa

**Foco:** Integração de **Flexbox dentro de um Grid**.

**Técnicas utilizadas:**
- Estrutura principal em **CSS Grid** (`1fr 280px`)
- Componentes internos organizados com **Flexbox** (Header, Sidebar, etc.)

---

# 🛠️ Tecnologias Utilizadas

- **HTML5** → Estruturação semântica  
- **CSS3**

### Principais conceitos aplicados:

- **Flexbox**  
  Para alinhamentos **unidimensionais**.

- **CSS Grid**  
  Para **layouts bidimensionais complexos**.

- **Variáveis e Seletores Avançados**  
  Para estilização **modular e reutilizável**.
